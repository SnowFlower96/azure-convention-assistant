# 템플릿 적용 범위
- 주석만 해당

# 체크리스트
- 필드, 생성자, 메소드 주석간 줄바꿈은 2라인
- 서비스/repository는 인터페이스를 상속하여 작성
- javadoc 주석은 어노테이션 위쪽
- 클래스/함수 수정시 @History에 새로운 라인 추가
- 클래스 내 상단과 하단은 줄바꿈 1라인
- 클래스 내 함수의 주석은 간략하게
- 코드 수정시 라인별 주석은 수정한 라인의 바로 위에

# 템플릿

 *                   - {1~2문장 정도의 간단한 수정내용}
